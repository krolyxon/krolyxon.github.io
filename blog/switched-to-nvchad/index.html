I officially switched to NvChad#

Throughout my journey of using Neovim, i always avoided using a pre-built config or any distribution of neovim such as LunarVim or AtroVim. I felt pride in building my own configuration. But, after spending countless amount of painful hours fixing things manually when some plugin decides to make some breaking changes out of nowhere. I finally decided i would just switch to a pre-built config like Lazyvim or NvChad.

Initially i tried, Lazyvim but i didn’t like it that much so i ended up using NvChad and to be honest its been a fun experience, everything just works out of the box. My muscle memory goes well with this configuration, since the configuration i used previously, which i made myself was very similar and inspired by NvChad.

The configuration is hosted on krolyxon/nvim and the old configuration is available in the same repo in a different branch.
